# ![Pickup Animations](https://github.com/Additional-Mods/pickupanimations/blob/main/mod_page_assets/title.png?raw=true)
Pickup Animations is a small, client-side QOL mod that adds a subtle pickup animation when you pick up an item in your inventory (or any other GUI).
